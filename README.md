# ansible-role-mariadb

Ansible role to manage databases using MariaDB

## Requirements

* Ansible version 2.*

## Role Variables

You can override all the variables in `defaults/main.yml`.

## Example Playbook

    - hosts: servers
      roles:
         - { role: monsieurbiz.mariadb }
